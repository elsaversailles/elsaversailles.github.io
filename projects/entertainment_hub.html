Turn Your Old Laptop/PC Into an Ad-Free Entertainment Hub

Here, I designed various projects from network engineering to MVP development. Leveraged various analytic software tools on creating the final design and construction.

Step 1:Download Lubuntu

  1. On your working computer, open this link: Lubuntu 24.04 LTS download page
  2. Scroll down until you see the download for Lubuntu 24.04 LTS (Long Term Support). Always choose the LTS version because it's stable and supported for a long time.
  3. Click the download link and wait until the .iso file finishes downloading. This file is like a “digital installer CD” for Lubuntu.

Step 2:Download and Install Ventoy (to make a bootable USB drive)

  1. Go to Ventoy download page.
  2. Download the Windows ZIP file (choose the latest release).
  3. After download, right-click the ZIP file and select Extract All.
  4. Open the extracted folder.
  5. Double-click on Ventoy2Disk.exe (it will open a small window).
  6. Insert an empty USB drive (at least 8 GB). ⚠️ Warning: This will erase everything on the USB drive! Back up your files first.
  7. In Ventoy, select your USB drive.
  8. Click Install. Confirm when it asks “Are you sure?”. Now your USB stick is a special Ventoy boot drive.

Step 3:Copy Lubuntu to the Ventoy Drive

  1. Find the Lubuntu .iso file you downloaded in Step 1.
  2. Copy (drag and drop) the .iso file directly onto the Ventoy USB drive.
  3. You can add more .iso files later if you want, but for now just Lubuntu.

Step 4:Boot Your Old Laptop/PC from the Ventoy USB Drive

  1. Plug the Ventoy USB drive into the old laptop/PC.
  2. Turn it on, and immediately press the key for Boot Menu/UEFI (varies by computer)
  3. In the boot menu, select your USB drive (Ventoy).

Step 5:Start Lubuntu Installer

  1. When Ventoy loads, you'll see a menu.
  2. Select the Lubuntu 24.04 .iso you copied earlier:
  3. Wait for it to boot, then click Install Lubuntu.

Step 6:Installation Setup

  1. Choose your language.
  2. Set your time zone/location (this makes the clock correct).
  3. Choose your keyboard layout (usually English US).
  4. When asked about installation type, choose Minimal Installation. (This keeps things lightweight since we only need Firefox.)

Step 7:Partition Setup

  1. When asked about installation type:
    • Choose Erase Disk and Install Lubuntu (do NOT dual-boot).
    • This makes startup faster and simpler. ⚠️ This will delete everything on the old computer's hard drive.
  2. Lubuntu will automatically handle swap space (virtual memory).

Step 8:Create a User Account

  1. Enter your name, computer name, and username.
  2. Set a password (write it down somewhere).
  3. Check the option: Log in automatically (no password prompt at boot).

Step 9:Final Confirmation

  1. Review the summary page.
  2. If everything looks right, click Install Now.
  3. Wait. This may take 15–30 minutes.
  4. When it finishes, it will ask you to remove the USB drive and press Enter.

Step 10:Install Firefox

  1. After restart, click the menu button (bottom left corner).
  2. Go to System Tools → QTerminal.
  3. In the terminal window, type: sudo apt install firefox
  4. Press Enter, type your password, then press Y when asked.
  5. Wait for Firefox to finish installing.

Step 11:Install Ublock Origin (Ad Blocker)

  1. Open Firefox.
  2. Click the menu button (three horizontal lines in the top-right).
  3. Select Add-ons and themes.
  4. In the search box, type: Ublock Origin.
  5. Click the first result and press Add to Firefox.
    • This blocks almost all ads automatically.

Step 12:Make Firefox Start Automatically on Boot

  1. Open the menu button again (bottom left).
  2. Go to Preferences → LXQt Settings → Session Settings.
  3. In the Autostart tab, click Add.
    • Paste this command: /usr/bin/firefox --kiosk https://cure-energy-54986886.figma.site/
  4. Save and close.
  5. Restart your computer, Firefox should now open automatically at startup.
    6. You can change the link on the command argument to your own UI

Step 13:Enjoy Your New Ad-Free Entertainment Hub

  1. Your old laptop/PC now works like a dedicated streaming box.
  2. For best experience, use a 2.4 GHz wireless remote with air-mouse.
    • This remote has:
      • A motion-controlled mouse
      • A keyboard on the back
      • Shortcut buttons for TV control
  3. Plug in your TV with HDMI, sit back, and enjoy!
